用代码实现就是：
```python
def minDistance(s1: str, s2: str) -> int:

    def dp(i, j):
        if i == -1: return j + 1
        if j == -1: return i + 1

        if s1[i] == s2[j]:
            return dp(i - 1, j - 1) 
        return min(
            dp(i, j - 1) + 1,   # 插入
            dp(i - 1, j) + 1,   # 删除
            dp(i - 1, j - 1) + 1 # 替换
        )
    return dp(len(s1) - 1, len(s2) - 1)
```

怎么定义dp_table呢
应该定义为
`把s1[0:i]变成s2[0:j]最少需要dp_table[i+1][j+1]步` 这里+1是因为需要补上空字符串
所以说初始条件应该是
`dp[0][j] = j, dp[i][0] = i`
