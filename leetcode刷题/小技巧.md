创建最小堆
```python
import heapq
pq = []
heapq.heappush(pq, (head.val, seq, head)) #进入
node = heapq.heappop(pq)[2] #推出
```
这里中间加个seq的原因是防止第一个元素head.val相等的时候比较第二个head会导致出错，所以需要加一个seq作为head.val相等的情况的避免。
